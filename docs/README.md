# Huddle Documentation

Documentation is organized by responsibility.

- 00-source-of-truth: source precedence and visual corpus policy.
- 01-product: product domains and scope.
- 02-architecture: ADRs, entities, routes and repository layout.
- 03-security-access: authorization and capability matrices.
- 04-api: canonical API contracts.
- 05-execution: roadmap and delivery workflow.
- 06-operations: environments, launch readiness and production operations.
- execution: task handoff records.
- 99-archive: superseded or reference-only material.
