# Huddle Documentation

Documentation is organized by responsibility, not by generation date.

- `00-source-of-truth/` — source precedence and visual corpus policy.
- `01-product/` — product scope and domain boundaries.
- `02-architecture/` — ADRs, entity catalog, routes and repository layout.
- `03-security-access/` — RBAC, capabilities and authorization policy.
- `04-api/` — canonical API contracts.
- `05-execution/` — roadmap, Sprint 0 source and delivery workflow.
- `06-operations/` — launch readiness and production operations.
- `execution/` — task handoff records.
- `99-archive/` — superseded/reference-only material.
