# Source of Truth

Priority when sources conflict:

1. Accepted ADRs and canonical engineering documentation.
2. The approved Figma master and its recorded canonical node IDs for visual decisions.
3. Actual merged repository implementation.
4. Locked brand documentation and approved repository assets.
5. The screen-route registry and canonical visual corpus.
6. Set55 flow maps.
7. Sprint 0 source where not superseded.
8. Older reference designs.

The visual corpus is stored under `assets/screens/` with stable set numbers and original filenames.

Visual implementation is governed by `design-source-of-truth.md` and `docs/01-product/screen-route-registry.csv`. PNG files are references unless an approved Figma node is recorded.
