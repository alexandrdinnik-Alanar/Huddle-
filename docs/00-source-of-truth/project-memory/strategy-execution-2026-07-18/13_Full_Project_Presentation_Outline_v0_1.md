# Huddle Full Project Presentation — Outline v0.1

Date: 2026-07-18  
Status: structure draft  
Source: Huddle Master Project Bible v0.2  
Target length: 45–55 slides  
Audience: team, advisors, strategic partners, early investors, public-value partners  
Design source: final Huddle Brand Guidelines only

## Purpose

This is the large visual presentation of Huddle. It is not the short public deck and not the investor pitch deck.

Its purpose is to show the full project clearly and visually:

- what Huddle is;
- why it matters;
- who it serves;
- how the ecosystem works;
- what we build first;
- how Oslo validation works;
- how the business and public-value logic connect;
- what team, funding and execution path are required.

## Design principle

The presentation should feel:

- warm;
- premium;
- human;
- calm;
- family-safe;
- modern;
- visually structured;
- not overloaded with text.

Use only approved Huddle brand colours, logo, icon and visual style.
