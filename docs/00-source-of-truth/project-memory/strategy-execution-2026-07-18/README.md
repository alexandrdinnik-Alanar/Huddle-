## Базовое правило

Ни один внешний документ Huddle не считается готовым, пока не прошёл:

- claims/evidence check;
- finance check, если есть деньги, бюджет, revenue или funding ask;
- legal/privacy/child-safety check, если документ касается детей, семей, данных, safety, муниципалитетов или marketplace;
- founder approval.
