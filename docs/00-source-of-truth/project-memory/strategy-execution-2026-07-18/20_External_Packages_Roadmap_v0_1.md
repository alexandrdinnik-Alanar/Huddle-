# Huddle External Packages Roadmap v0.1

Date: 2026-07-18  
Status: working roadmap  
Source: Master Project Bible v0.3 + Document Map v1  
Purpose: define the external document packages that will be created after the Master Bible and presentation copy.

## Rule

No external package is ready to send until:

- source claims are checked;
- financial assumptions are reconciled;
- legal/privacy/child-safety wording is reviewed;
- founder approval is recorded;
- final Huddle brand is applied.

- advisors who can introduce investors.

Show why Huddle can become a large, investable company and why the Oslo-first evidence-gated strategy is credible.
