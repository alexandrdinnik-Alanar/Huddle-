# Product

Huddle product scope is split into explicit domains: Identity and Access, Family, Connections and Matching, Community, Groups, Messaging, Listings, Catalog and Partner Feed, Orders and Fulfillment, Payments and Settlement, Events and Activities, Services and Experts, Advertising, Municipality, Trust and Safety, Notifications, Support, Analytics and Reporting, Privacy and Governance, and Platform Operations.

Product behavior must be implemented through the canonical routes, entity catalog, capability matrix and API contracts in the neighboring documentation sections.
