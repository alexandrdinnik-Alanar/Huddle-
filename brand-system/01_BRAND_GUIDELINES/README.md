# 01 — Brand Guidelines

## Structure
- `01_LAYERED_SOURCE/` — primary editable Brand Guidelines source. Text, fonts, blocks, positions and vector shapes should remain editable.
- `02_APPROVED_REFERENCE_PNG/` — approved visual design reference. Use for fidelity checks; do not treat as editable source.
- `03_REFERENCE_PDF/` — ordered reference/export.
- `04_CANVA_IMPORT/` — Canva-oriented import source.

## Rule
When editing, preserve the approved design shown in reference PNG pages while rebuilding text and shapes as editable layers. Do not duplicate pages by content.
