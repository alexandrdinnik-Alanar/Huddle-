# Huddle Brand Guidelines — Layered Source Rebuild

This folder contains a rebuilt editable source deck for the approved Huddle Brand Guidelines.

## Main file
- `Huddle_Brand_Guidelines_LAYERED_SOURCE_Canva_Figma.pptx`

## What is editable
- Text layers
- Typography settings
- Shapes
- Cards
- Rules / dividers
- Labels
- Background forms
- Network / node motifs
- Construction geometry
- UI / campaign / layout blocks
- Object positions and sizes

## What remains raster
- Approved logo PNG placements are replaceable identity assets. Use the master assets from `02_MASTER_IDENTITY` for final production.
- Photorealistic renders, website screenshots and lifestyle scenes remain image objects. They can be moved, cropped, replaced or removed, but not edited as vectors.

## Reference policy
The original approved PNG pages are included in `approved_png_references`. They are visual reference only and should not be used as locked slide backgrounds if full editability is required.

## Canva
Upload the PPTX through: Create design → Import file → Upload PPTX.

## Figma
Use this PPTX as a source deck and import SVG elements from the production package for deeper vector editing.
