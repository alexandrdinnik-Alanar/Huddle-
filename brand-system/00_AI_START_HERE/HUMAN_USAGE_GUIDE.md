# HUMAN USAGE GUIDE

## Canva
1. Import the layered PPTX from `01_BRAND_GUIDELINES/01_LAYERED_SOURCE/`.
2. Edit text, fonts, positions, blocks and shapes.
3. Upload SVG assets from `03_EDITABLE_GRAPHIC_SYSTEM/`.
4. Use approved identity assets from `02_MASTER_IDENTITY/`.
5. Use approved PNG pages only as visual reference, not as flattened editable content.

## Figma
1. Import SVG assets directly.
2. Import design tokens from `04_DESIGN_TOKENS/` using a compatible token workflow.
3. Use PDF/SVG identity masters from `02_MASTER_IDENTITY/`.
4. Keep approved PNG guideline pages open as visual reference.

## PowerPoint
Use the layered PPTX as the primary editable Brand Guidelines source.

## Print
Use approved PDF identity masters and print assets. Validate converted SVG identity exports against the PDF masters before final production.
