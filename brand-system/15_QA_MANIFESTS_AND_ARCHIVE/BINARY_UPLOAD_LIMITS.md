# Binary Upload Limits

The connected GitHub tool can create UTF-8 text files and text-based editable assets through the repository contents workflow.

The following binary categories remain pending because the connector does not provide a direct local-file-to-repository binary upload action in this session:
- PPTX
- PDF
- PNG
- JPG

The 111 pending binary files are enumerated by path and size in `NOT_UPLOADED_BINARY_FILES.md` / transfer manifests. Missing binaries must not be regenerated by AI.
