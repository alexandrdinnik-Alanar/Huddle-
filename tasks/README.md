# Huddle Task Contracts

Each implementation task has a stable task ID and lives under a phase folder.

Rules:
- one task ID per branch;
- read dependencies before editing;
- follow allowed and forbidden file scopes in the task;
- run required checks;
- write `docs/execution/<TASK-ID>/HANDOFF.md`;
- do not start the next task automatically.
