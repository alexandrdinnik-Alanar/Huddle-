## Required route groups

```text
src/app/
├── (public)/
├── (auth)/
├── (app)/
├── (admin)/
└── api/
```

URL namespace rules:

- public pages remain outside protected URL prefixes;
- authenticated product routes use `/app/**`;
- privileged operations use `/admin/**`;
- route groups do not change URL paths;
- no guards are implemented before S0-011.

## Placeholder contract

Each placeholder must visibly show:

- screen name;
- route pattern;
- priority;
- temporary Sprint 0 placeholder notice;
- exact note: `Feature implementation is out of scope for Sprint 0.`

The route metadata must be centralized so placeholder labels and priorities do not drift across page files.
