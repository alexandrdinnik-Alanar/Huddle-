## Required implementation

1. Initialize a React 19.2 baseline application using Next.js 16.2.x App Router and TypeScript strict mode.
2. Use pnpm and commit `pnpm-lock.yaml`.
3. Create scripts: `dev`, `build`, `lint`, `typecheck`, `test`, `test:e2e`, `format:check`.
4. Create `src/app`, `src/components`, `src/modules`, `src/lib`, `prisma`, `tests` and required config files.
5. Add `.env.example` with placeholders only.
6. Preserve `AGENTS.md`, `CLAUDE.md`, `.ai/HUDDLE_RULES.md`, `tasks/` and `docs/`.
7. Add `docs/02-architecture/runtime-baseline.md` with exact resolved dependency versions.
8. Do not implement product features, authentication flows, business schema or business routes in this task.

- business routes not required for foundation;

- production credentials or secrets;
- unapproved provider commitments;
- unrelated roadmap tasks.

## React implementation contract

- React UI only.
- Next.js App Router only.
- Prefer Server Components by default.
- Use Client Components only for actual browser interaction or state.
- Keep business rules out of React components.
- Preserve keyboard accessibility for interactive UI.
