## Contents

## Governance

These files are design and prototype references.

They are not automatically:

- production application routes;
- approved Figma screens;
- production React components;
- implementation specifications.

Canonical production design decisions must be approved in the Huddle Figma master and recorded in the screen-route registry before implementation.

Do not delete, rename, deduplicate, or modify original source files without a separate approved task.
