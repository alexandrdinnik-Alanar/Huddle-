# Huddle Brand Assets

This directory is reserved for the approved Huddle Brand Guidelines and Logo Master Package.

## Canonical packages

- `guidelines/Huddle_Brand_Guidelines.zip`
- `guidelines/Huddle_Family_Brand_Guidelines.pdf`
- `logo-master/Huddle_Family_Logo_Master_Package.zip`

## Runtime exports

Production web exports must be copied from the approved Logo Master Package into `public/brand/` using these names:

- `huddle-family-logo.png`
- `huddle-family-logo.webp`
- `huddle-wordmark.png`
- `huddle-icon.png`
- `icons/favicon.ico`
- `icons/apple-touch-icon.png`
- `icons/icon-192.png`
- `icons/icon-512.png`

Do not substitute generated, redrawn, cropped, recolored, or automatically traced logo versions.

## Approved palette

- Warm Cream `#F6F0E8`
- Sand Beige `#E8D9C9`
- Copper Clay `#C98A5C`
- Soft Olive `#8A9076`
- Graphite Taupe `#7C6D63`
- White `#FFFFFF`

`Closer by Design` is retired and must not be used in new production assets.

## Large-file note

GitHub rejects individual files larger than 100 MB without Git LFS. The complete uploaded source bundle contains nested archives above that limit. Store distributable subpackages individually or configure Git LFS before committing oversized archive files.
